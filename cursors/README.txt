Custom cursor for TarkLend site
================================

To use the Sanrio Cinnamoroll cursor on the website:

1. Get the cursor image from your Sweezy Cursors pack (e.g. from sweezy-cursors.com).
2. Add a PNG version here:
   - cursor.png  (recommended: 32x32 px, for normal cursor)
   Optional: cursor-pointer.png for links/buttons.

The site CSS looks for:
  cursors/cursor.png
  cursors/cursor.cur (optional, for older browsers)

If no file is present, the browser will use the default arrow.
